🚀 Employee Management System

A Fullstack Employee Management System built using Spring Boot (Gradle), React, and MySQL.
The application allows efficient management of employee records with secure REST APIs and a responsive frontend interface. 💼📊

📌 Features

➕ Add new employees

📋 View all employees

✏️ Update employee details

❌ Delete employees

🔍 Search & manage records efficiently

🌐 RESTful API integration

🎨 Responsive UI with modern frontend

🏗️ Tech Stack
🔹 Backend

Spring Boot

Gradle

Spring Data JPA

MySQL

🔹 Frontend
